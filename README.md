# Sha512_256

SHA512/256 hashing function.


```elixir
iex> :sha2.hexdigest512_256 ""
<<198, 114, 184, 209, 239, 86, 237, 40, 171, 135, 195, 98, 44, 81, 20, 6, 155,
  221, 58, 215, 184, 249, 115, 116, 152, 208, 192, 30, 206, 240, 150, 122>>
iex> Base.encode16 v()
"C672B8D1EF56ED28AB87C3622C5114069BDD3AD7B8F9737498D0C01ECEF0967A"
```

```elixir
iex> Sha512_256.hash "hello world"
<<10, 197, 97, 250, 200, 56, 16, 78, 63, 46, 74, 209, 7, 180, 190, 227, 233, 56,
  191, 21, 242, 177, 95, 0, 156, 204, 205, 97, 169, 19, 240, 23>>
```
